Demo 5 — Lit + Bootstrap-Adapter

Dieselben Components wie Demo 4. Nur ein zusätzliches Stylesheet (simplecmp/styles/bootstrap.css) und schon übernimmt das UI Bootstraps Farben, Radius, Schriftart.

So funktioniert's: Der Adapter mappt --simplecmp-color-primary auf --bs-primary, --simplecmp-radius auf --bs-border-radius, usw. Custom Properties durchqueren die Shadow-DOM-Grenze automatisch — kein JS, kein Fork, kein Theming-Framework.
