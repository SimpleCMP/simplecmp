Demo 4 — neue Lit-UI REQ-14

Dieselbe Funktionalität wie Demo 1, aber mit der neu gebauten Lit-basierten UI (REQ-14, ADR-0007). Banner, Modal, Trigger sind Web Components mit Shadow DOM — keine Klaro-Preact-Reste mehr.

  • <simplecmp-banner>: bottom-right, REQ-1 Datenschutz/Impressum-Links separat
  • <simplecmp-modal>: native <dialog>, Focus-Trap und Esc von der Plattform (REQ-6)
  • <simplecmp-trigger>: Floating-Button (REQ-4)
  • Hybride Shadow/Light-DOM via mode="light" — siehe Demo 5 mit Bootstrap-Theme
